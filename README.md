# PBoard
Текстовый процессор на C#

<img width="1432" height="804" alt="Preview" src="https://github.com/user-attachments/assets/444584d6-6f00-4825-b649-90ad8ed79c65" />

## Функционал
- Текст размещается в текстовых блоках;
- Блоки можно настраивать: цвет, границы, фон;
- Текст можно стилистически имзенять: направление текста, толщина, шрифт, высота, цвет, фон;
- На холст можно разместить изображения;
- Экспорт и импорт.
